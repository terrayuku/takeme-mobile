# TakeMe Mobile - Documentation Summary

## Project Overview

TakeMe Mobile is a crowdsourced mobile application designed to help commuters in South Africa find taxi/minibus routes by sharing and viewing photos of taxi destination signs. The app addresses a critical pain point in the informal transportation sector where millions of daily commuters struggle to identify which taxis go to their desired destinations.
