# TakeMe Mobile - Project Documentation

## Overview
TakeMe is a mobile application designed to help commuters in South Africa find taxi/minibus routes by crowdsourcing and sharing photos of taxi destination signs. The app enables users to search for routes, view taxi signs, and contribute by uploading photos of signs they encounter.
